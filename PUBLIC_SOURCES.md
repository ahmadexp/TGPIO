# Public Sources

This project is based on original implementation work by Ahmad Byagowi and
public Linux/kernel interface information.

Public references used for validation and compatibility:

- Linux `pps_gen_tio.c`, the upstream Intel PPS/TIO generator driver:
  <https://codebrowser.dev/linux/linux/drivers/pps/generators/pps_gen_tio.c.html>
- Public Linux kernel mailing-list discussion for Intel PMC TGPIO support:
  <https://lkml.iu.edu/1907.2/00067.html>
- Public Linux kernel mailing-list discussion containing TGPIO input/output
  implementation context:
  <https://lkml.iu.edu/1907.2/00613.html>

No non-public source archives, patches, or documentation are included in this
repository.

If adding new material, use public sources or original code only. Do not copy
text, comments, patches, or source files from non-public materials.
